Guida Utente Harzafi Cloud

La tua bussola per navigare ed utilizzare al meglio tutte le funzionalità sicure e intuitive di Harzafi Cloud. Inizia il tuo viaggio verso una gestione dei file semplice ed efficiente!

Benvenuto/a in Harzafi Cloud!

Harzafi Cloud è il tuo spazio digitale personale, sicuro e affidabile, progettato per semplificare la gestione dei tuoi file. Che tu abbia bisogno di archiviare documenti importanti, foto di famiglia o progetti di lavoro, Harzafi Cloud ti offre gli strumenti per farlo con facilità e tranquillità. Il nostro impegno è fornirti un servizio cloud gratuito (piano base da 5 GB), potente e intuitivo.

Questa guida ti accompagnerà passo dopo passo attraverso tutte le funzionalità principali di Harzafi Cloud, dalla creazione del tuo account alla gestione avanzata dei file. Se sei pronto, iniziamo questo viaggio insieme!

Primi Passi: Creazione e Accesso

1. Creazione del Tuo Account Gratuito

Iniziare con Harzafi Cloud è un processo rapido e semplice. Ecco come creare il tuo account in pochi minuti:

  1. Visita la nostra pagina di registrazione (File: registrazione.html).
  2. Compila i dati personali: Ti chiederemo il tuo nome, cognome e data di nascita. Ricorda, devi avere almeno 18 anni per registrarti, come indicato dal controllo di validità nel modulo.
  3. Indirizzo Email e Password: Fornisci un indirizzo email valido (sarà il tuo username per il login) e crea una password sicura. La pagina di registrazione (registrazione.html) presenta un indicatore di robustezza della password e i criteri da rispettare:
    • Almeno 8 caratteri
    • Almeno una lettera maiuscola
    • Almeno una lettera minuscola
    • Almeno un numero
    • Almeno un carattere speciale (es. !@#$%)
  4. Immagine del Profilo (Opzionale ma consigliato!): Personalizza il tuo account caricando un'immagine del profilo. Puoi trascinare un file immagine nell'area designata o cliccare su "Scegli File...". Supportiamo formati comuni come PNG, JPG, GIF, con una dimensione massima di 5MB. Dopo aver selezionato l'immagine, potrai utilizzare il nostro strumento di ritaglio e aggiustamento per perfezionarla (basato su Cropper.js).
  5. Località: Seleziona il tuo Paese ed eventualmente la Regione/Stato dai menu a tendina. La selezione del paese abiliterà il menu delle regioni, se applicabile.
  6. Conferma: Dopo aver compilato tutti i campi obbligatori e, se lo desideri, l'immagine del profilo, clicca sul pulsante "Registrati". Il server (server.js) verificherà i dati (ad es. se l'email è già in uso).
  7. Se la registrazione va a buon fine, visualizzerai un messaggio di successo e sarai pronto per accedere.

Consiglio per la Password: Utilizza un gestore di password per creare e memorizzare password complesse e uniche per ogni servizio, incluso Harzafi Cloud. Non usare mai la stessa password su più siti!

2. Accedere al Tuo Account

Una volta che il tuo account è stato creato con successo, accedere è semplice:

  1. Naviga alla pagina di login (File: accedi.html).
  2. Inserisci l'indirizzo email e la password che hai utilizzato durante la registrazione. Entrambi i campi sono obbligatori e validati.
  3. Opzione "Ricordami": Se stai utilizzando un dispositivo privato e sicuro, puoi spuntare la casella "Ricordami". Questo, tipicamente, imposta un cookie di sessione più duraturo (la logica esatta dipende dall'implementazione del server server.js e dalla configurazione di express-session). È sconsigliato su computer condivisi.
  4. Clicca sul pulsante "Accedi". Il server (server.js) verificherà le tue credenziali confrontando la password fornita con l'hash memorizzato (utilizzando bcryptjs).

Se le credenziali sono corrette, sarai reindirizzato alla tua Dashboard personale di Harzafi Cloud (Harzafi Cloud.html).

Password Dimenticata? Nessun Problema!

Se hai dimenticato la tua password, sulla pagina di login, clicca sul link Password dimenticata? (File: password_dimenticata.html).

  1. Ti verrà chiesto di inserire il tuo indirizzo email associato all'account.
  2. Una volta inviata l'email, il sistema (attualmente simulato in password_dimenticata.html) ti presenterà un campo per inserire un codice OTP (One-Time Password) a 6 cifre che "sarebbe stato inviato" alla tua email.
  3. Inserisci il codice OTP. Se corretto (simulazione), ti verrà data la possibilità di impostare una nuova password (questa parte non è implementata nella demo).
Nota sulla Demo: La funzionalità di invio email e reimpostazione password effettiva non è implementata nel server.js fornito. La pagina password_dimenticata.html simula il processo lato client.

Gestione Dettagliata dei File

Harzafi Cloud ti offre strumenti intuitivi per gestire i tuoi file in modo efficiente. Tutte le operazioni sui file avvengono principalmente nel riquadro "I Tuoi File" sulla dashboard.

Importante Riguardo la Versione Demo: Come menzionato, nella versione attuale (Harzafi Cloud.html), i file che carichi sono gestiti utilizzando localStorage nel tuo browser. Questo significa che:
  • I file sono memorizzati localmente sul tuo computer, all'interno del browser che stai utilizzando.
  • Sono accessibili solo da quella specifica sessione del browser e su quel dispositivo.
  • Non vengono inviati o archiviati su un server cloud remoto persistente.
  • L'eliminazione dei dati del browser, l'utilizzo di una finestra di navigazione in incognito, o l'accesso da un altro browser/dispositivo comporterà la "perdita" di questi file dalla vista della dashboard.
Questa è una limitazione della demo attuale; un servizio cloud completo garantirebbe la persistenza dei dati su server remoti.

1. Caricare File sul Tuo Spazio Cloud

Aggiungere file al tuo Harzafi Cloud è un'operazione semplice. Puoi farlo in due modi principali:

  1. Tramite il Pulsante "Carica File":
    • All'interno del riquadro "I Tuoi File" sulla tua dashboard, individua e clicca il pulsante Carica File. Se il tuo spazio file è vuoto, potresti vedere un pulsante alternativo come Aggiungi File.
    • Questa azione aprirà la finestra di dialogo standard del tuo sistema operativo per la selezione dei file.
    • Naviga tra le tue cartelle locali, seleziona uno o più file che desideri caricare, e conferma la selezione.
    • I file scelti verranno processati (localmente nella demo) e appariranno come nuovi elementi nella griglia dei file sulla tua dashboard.
  2. Funzionalità Drag & Drop (Tipica dei Sistemi Cloud):

    Molti servizi cloud moderni permettono di trascinare file direttamente dal desktop o da una cartella del computer sull'interfaccia web per caricarli. Sebbene questa specifica interazione non sia esplicitamente codificata per la griglia dei file in Harzafi Cloud.html, la pagina di registrazione (registrazione.html) implementa il drag & drop per l'immagine del profilo, suggerendo che tale intuitiva funzionalità è considerata nel design complessivo di Harzafi Cloud.

Tipi di File Supportati e Anteprime:

Puoi caricare qualsiasi tipo di file. Per i file immagine (come .png, .jpg, .gif) di dimensioni contenute (sotto i 5MB, per motivi di performance del localStorage e della generazione di dataURL), Harzafi Cloud cercherà di mostrare un'anteprima visiva direttamente nella griglia dei file. Per altri tipi di file, o immagini più grandi, verrà visualizzata un'icona generica rappresentativa del tipo di file (es. per PDF, per documenti Word, per archivi ZIP, ecc.).

2. Visualizzare e Organizzare i Tuoi File

  • Layout a Griglia Intuitivo: I tuoi file sono presentati in una chiara e ordinata vista a griglia, rendendo facile localizzare ciò che cerchi.
  • Dettagli Rapidi del File: Per ogni file nella griglia, puoi visualizzare immediatamente:
    • Il nome completo del file (con tooltip al passaggio del mouse se il nome è troppo lungo).
    • L'icona o l'anteprima dell'immagine.
    • La dimensione del file, formattata in modo comprensibile (es., 1.2 MB, 250 KB).
  • Ordinamento e Filtri (Funzionalità Avanzate Future): Sebbene non presenti nella demo attuale, versioni future di Harzafi Cloud potrebbero includere opzioni avanzate per ordinare i file (per nome, data di modifica, dimensione, tipo) e filtri per restringere la visualizzazione a specifici tipi di file o criteri di ricerca.
  • Organizzazione in Cartelle (Funzionalità Avanzate Future): La capacità di creare, rinominare ed eliminare cartelle per raggruppare logicamente i tuoi file è una caratteristica fondamentale pianificata per migliorare l'organizzazione del tuo spazio cloud.

3. Anteprime dei File Dettagliate

Come menzionato, Harzafi Cloud si impegna a fornire anteprime utili. Nella versione demo (Harzafi Cloud.html), le anteprime dirette nella griglia sono principalmente per file immagine di piccole dimensioni, grazie alla generazione di `dataUrl` via JavaScript. La pagina principale del servizio (index.html) promuove "Anteprime Avanzate", indicando l'intenzione di supportare anteprime integrate per una gamma più vasta di formati (documenti, PDF, video, audio) in una versione completa, permettendoti di visualizzare i contenuti senza doverli necessariamente scaricare.

4. Condivisione Sicura dei File (Funzionalità Chiave Futura)

La condivisione è essenziale. La pagina promozionale index.html evidenzia "Condivisione Sicura" come una delle funzionalità principali di Harzafi Cloud. Questo implica che in una versione completa del servizio potrai:

  • Generare link condivisibili per i tuoi file e cartelle.
  • Proteggere questi link con password.
  • Impostare date di scadenza per i link di condivisione.
  • Definire permessi specifici per gli utenti con cui condividi (es. sola lettura, modifica).

Queste importanti funzionalità di condivisione non sono implementate nella dashboard demo attuale (Harzafi Cloud.html).

5. Eliminare File dal Tuo Spazio

Rimuovere file che non ti servono più è un'operazione semplice e diretta:

  1. All'interno della griglia dei file, individua il file che desideri eliminare.
  2. Porta il cursore del mouse sopra l'elemento del file. Noterai che appare un piccolo pulsante rosso con un'icona a forma di croce () nell'angolo in alto a destra dell'elemento del file.
  3. Clicca su questo pulsante di eliminazione.
  4. Per prevenire eliminazioni accidentali, il sistema ti chiederà una conferma esplicita: "Sei sicuro di voler rimuovere "[nome file]"? L'azione è limitata a questa sessione del browser.".
  5. Se confermi la tua intenzione, il file sarà rimosso dalla griglia e, nella demo attuale, dal LocalStorage del tuo browser.
Funzionalità "Cestino" e Versionamento (Pianificate):

Come menzionato in index.html ("Versionamento File"), Harzafi Cloud mira a offrire funzionalità avanzate come il versionamento dei file (per ripristinare versioni precedenti) e un "Cestino" da cui recuperare file eliminati accidentalmente entro un certo periodo. Queste funzionalità non sono presenti nella demo attuale.

3. Logout Sicuro dal Tuo Account

È di cruciale importanza disconnettersi correttamente da Harzafi Cloud, specialmente se accedi al servizio da computer condivisi o reti pubbliche, per proteggere la privacy e la sicurezza dei tuoi dati.

  1. Individua e clicca sul tuo avatar (o le tue iniziali) nell'angolo in alto a destra dell'header della pagina.
  2. Si aprirà un menu a tendina. Seleziona l'opzione Logout.
  3. Questa azione invierà una richiesta al server (server.js) per distruggere la tua sessione corrente. Il server, a sua volta, pulirà il cookie di sessione (connect.sid) dal tuo browser.
  4. Verrai automaticamente reindirizzato alla pagina principale di Harzafi Cloud (index.html), confermando che la disconnessione è avvenuta con successo.
Non chiudere semplicemente la scheda del browser! Chiudere la scheda o il browser senza effettuare il logout potrebbe non terminare immediatamente la tua sessione sul server, specialmente se hai selezionato "Ricordami". Usa sempre il pulsante Logout per la massima sicurezza.

Consigli Proattivi per la Sicurezza

In Harzafi Cloud, la sicurezza dei tuoi dati è una priorità assoluta. Come descritto ampiamente nella nostra sezione Sicurezza sulla pagina principale (index.html), implementiamo tecnologie e protocolli avanzati, tra cui:

  • Crittografia Robusta: Utilizzo di standard come AES-256 bit per i file archiviati (at rest) e durante il trasferimento (in transit).
  • Trasferimenti Sicuri: Comunicazioni tramite protocolli sicuri TLS/SSL.
  • Autenticazione Forte: Meccanismi robusti e il previsto supporto per l'Autenticazione a Due Fattori (2FA).
  • Infrastruttura Resiliente: Data center protetti e ridondanza dei dati.
  • Monitoraggio Continuo: Sistemi attivi 24/7 per rilevare e mitigare minacce.
  • Privacy Garantita: Policy trasparenti e nessun utilizzo dei tuoi file per scopi pubblicitari.

Tuttavia, la sicurezza è una responsabilità condivisa. Ecco alcuni consigli proattivi che puoi adottare per rafforzare ulteriormente la protezione del tuo account e dei tuoi dati:

Gestione delle Password

  • Crea Password Forti e Uniche: Utilizza password di almeno 12-16 caratteri, combinando lettere maiuscole, minuscole, numeri e simboli. Evita parole comuni, date di nascita o sequenze facili da indovinare. La pagina di registrazione (registrazione.html) ti aiuta con i criteri minimi.
  • Non Riutilizzare le Password: Usa una password diversa per Harzafi Cloud rispetto agli altri tuoi account online. Se una password viene compromessa su un altro sito, il tuo account Harzafi Cloud rimarrà protetto.
  • Utilizza un Gestore di Password: Considera l'uso di un gestore di password affidabile per generare e memorizzare password complesse in modo sicuro.
  • Cambia la Password Periodicamente: O immediatamente se sospetti che possa essere stata compromessa.

Autenticazione e Accesso

  • Abilita l'Autenticazione a Due Fattori (2FA): Non appena questa funzionalità sarà disponibile su Harzafi Cloud (come pianificato), abilitala. Aggiunge un ulteriore livello di verifica oltre alla tua password.
  • Logout Sistematico: Effettua sempre il logout dal tuo account Harzafi Cloud quando hai finito di usarlo, specialmente se accedi da computer pubblici, condivisi o reti Wi-Fi non protette.
  • Riconosci Tentativi di Phishing:
    • Sii scettico nei confronti di email, messaggi o chiamate non richieste che ti chiedono le tue credenziali di Harzafi Cloud o informazioni personali.
    • Harzafi Cloud non ti chiederà MAI la tua password via email o telefono.
    • Verifica sempre l'indirizzo del mittente e l'URL dei link prima di cliccare. Accedi a Harzafi Cloud digitando direttamente l'indirizzo nel browser o usando i tuoi segnalibri fidati.

Sicurezza del Dispositivo e della Rete

  • Mantieni Aggiornati Software e Dispositivi: Assicurati che il tuo sistema operativo, browser web e software antivirus/antimalware siano sempre aggiornati alle ultime versioni per proteggerti dalle vulnerabilità note.
  • Utilizza Reti Sicure: Evita di accedere a informazioni sensibili o al tuo account Harzafi Cloud quando sei connesso a reti Wi-Fi pubbliche o non protette, poiché potrebbero essere più vulnerabili alle intercettazioni. Se devi farlo, considera l'uso di una VPN (Virtual Private Network).
  • Attenzione ai Malware: Installa software solo da fonti attendibili e fai attenzione agli allegati email sospetti per evitare malware che potrebbero compromettere le tue credenziali o i tuoi file.
La tua Vigilanza è Fondamentale: Monitora regolarmente l'attività del tuo account (quando saranno disponibili i log di attività) e segnalaci immediatamente qualsiasi attività sospetta.

Risoluzione dei Problemi Comuni

Anche con un servizio intuitivo come Harzafi Cloud, a volte possono sorgere piccoli intoppi. Ecco una guida per risolvere i problemi più comuni che potresti incontrare.

Problemi di Accesso (Login e Registrazione)

  • Errore: "Email o password non corretti" (accedi.html)
    • Soluzione: Verifica attentamente che l'indirizzo email e la password inseriti siano corretti. Presta attenzione a maiuscole/minuscole, spazi accidentali e caratteri speciali. Assicurati che il tasto "Bloc Maiusc" non sia attivo. Se sei sicuro delle credenziali, prova a reimpostare la password.
  • Errore: "Utente già registrato con questa email" (durante la registrazione in registrazione.html)
    • Soluzione: Questo messaggio indica che un account con l'indirizzo email fornito esiste già. Prova ad accedere con quell'email. Se hai dimenticato la password, utilizza la funzione "Password dimenticata?" sulla pagina di login.
  • Errore: "Devi avere almeno 18 anni per registrarti" (registrazione.html)
    • Soluzione: Il sistema ha calcolato che la data di nascita inserita corrisponde a un'età inferiore ai 18 anni. Verifica la data inserita. Harzafi Cloud richiede che gli utenti abbiano almeno 18 anni.
  • Errore: "Password debole. Controlla i criteri." (registrazione.html)
    • Soluzione: La password che hai scelto non soddisfa i requisiti di sicurezza minimi. Assicurati che la tua password includa almeno 8 caratteri, una lettera maiuscola, una minuscola, un numero e un simbolo speciale. L'interfaccia di registrazione fornisce un feedback visivo sui criteri soddisfatti.
  • Non ricevo il codice OTP per la reimpostazione della password (password_dimenticata.html)
    • Soluzione (Demo): Nella versione demo attuale, il codice OTP viene visualizzato direttamente sulla pagina a scopo di test e non viene inviato via email. Assicurati di copiare correttamente quel codice.
    • Soluzione (Sistema Reale Futuro): Controlla la cartella Spam o Posta Indesiderata della tua casella email. Assicurati di aver inserito l'indirizzo email corretto. Attendi qualche minuto, a volte le email possono subire ritardi. Se il problema persiste, potrebbe esserci un'opzione per richiedere un nuovo codice.

Problemi con i File (Specifici della Versione Demo con LocalStorage)

  • I miei file non sono visibili dopo aver chiuso il browser o cambiato dispositivo.
    • Causa: Nella versione demo (Harzafi Cloud.html), i file vengono memorizzati nel LocalStorage del browser. Questa memoria è locale al browser e al dispositivo specifico.
    • Soluzione: Purtroppo, con l'attuale implementazione demo, i file non sono persistenti tra sessioni diverse su browser diversi o dopo la cancellazione dei dati del browser. Per accedere ai file, devi utilizzare lo stesso browser e profilo utente sul medesimo dispositivo in cui li hai caricati, senza aver cancellato i dati del sito.
  • Errore durante il caricamento di file di grandi dimensioni.
    • Causa: LocalStorage ha limiti di dimensione (generalmente 5-10 MB per dominio, a seconda del browser). File più grandi potrebbero non essere memorizzabili. Anche l'uploader dell'immagine del profilo in registrazione.html ha un limite implicito di 5MB per la gestione client-side.
    • Soluzione: Prova a caricare file più piccoli. In un servizio cloud completo, i limiti di dimensione dei file sarebbero significativamente più alti e gestiti lato server.
  • Le anteprime dei file non vengono visualizzate correttamente.
    • Causa: Nella demo, le anteprime dirette nella griglia sono generate via JavaScript principalmente per file immagine comuni e di dimensioni ridotte (che permettono la creazione di un `dataUrl` efficiently).
    • Soluzione: Per altri tipi di file, vedrai un'icona generica. Verifica che il file immagine sia in un formato supportato (PNG, JPG, GIF) e non eccessivamente grande.
  • Il pulsante "Elimina File" non funziona o il file riappare.
    • Causa: Potrebbe esserci un problema temporaneo con lo script JavaScript che gestisce la rimozione da LocalStorage, oppure un'interferenza da estensioni del browser.
    • Soluzione: Prova a ricaricare la pagina. Se il problema persiste, prova a disabilitare temporaneamente le estensioni del browser per vedere se una di esse sta causando un conflitto. In ultima istanza, puoi cancellare manualmente il LocalStorage per il sito tramite gli strumenti per sviluppatori del browser, ma questo eliminerà tutti i file "caricati" nella demo.
Strumenti per Sviluppatori del Browser: Se sei un utente tecnico, puoi usare gli strumenti per sviluppatori del tuo browser (solitamente accessibili con F12) per controllare la console JavaScript per eventuali messaggi di errore, o per ispezionare il LocalStorage e vedere cosa vi è memorizzato.
